# Auth

A simple auth middleware server.

# Usage

[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy)

or

Local:

```bash
# Configure your env
cp .env.example .env
npm i
npm start
# open localhost:3000
```

### Example use case

Authenticate using auth0 to access admin rethinkdb dashboard.
